# OLS — نظام التعلّم العُماني (Omani Learning System)

منصّة تعليمية تفاعلية للمنهج الدراسي العُماني من **الروضة إلى الصف الثاني عشر**.
تطبيق صفحة واحدة (SPA) ثابت + خادم Node بدون أي مكتبات خارجية — مبني على معمارية
RH ProjectHub المجرَّبة (يعمل على استضافة Node مشتركة رخيصة مثل Hostinger).

## الصفحات
- **الرئيسية** — لوحة معلومات بملخّصات وإحصاءات ووصول سريع.
- **المناهج** — الكتب مرتّبة حسب المستوى (روضة–12) والفصل الدراسي، بقارئ على شكل رفّ كتب.
- **المكتبة** — مكتبة المنهج العُماني الكاملة (٣٦٨ كتابًا مدرسيًا، صفوف ١–١٢) مصنّفة حسب
  الصف والفصل والمادة، مع بحث وزر تبديل بين **📖 الكتاب التفاعلي** (القارئ الرسمي من بوابة
  الوزارة) و**📄 PDF**، وفتح مباشر داخل الصفحة وملء الشاشة. يضاف إليها مرفقات المعلّم.
  الكاتالوج في `library-data.js`. **ملفات PDF (~18GB) غير مضمّنة**؛ لتفعيل خيار PDF ارفع مجلد
  المكتبة إلى الخادم تحت `library/` (أو حدّد مسارًا من ⚙️ في صفحة المكتبة) — الروابط التفاعلية
  تعمل دائمًا بدونها.
- **الحصص** — فيديو، صوت، ومستندات (PDF/Word/PowerPoint/Excel/صور) مصنّفة حسب الصف، مع
  عرض ملء الشاشة وعرض شرائح PowerPoint، والتنزيل والاستبدال والحذف (للمدير).
- **التمارين** — تدريبات مهارية تلقائية التصحيح (حساب، ترتيب حروف…)، بالإضافة إلى **🎨 لوحة
  تدريب A4** كاملة: كتابة ورسم وتلوين ومسح وتراجع، قوالب مسطّرة (عربي، إنجليزي بأربعة سطور،
  رياضيات بمربّعات، نقاط، خانات)، حفظ صورة وطباعة، وأداة **تتبّع الحروف/الكلمات المنقّطة**
  (يكتب المعلّم النص فيتحوّل إلى نموذج منقّط/مفرّغ قابل للتتبّع، بتحكّم في حجم الخط وعدد التكرار).
- **الاختبارات** — اختبارات تفاعلية تُجاب وتُصحّح فورًا مع تقرير أداء ومؤقّت.
- **النتائج** — تقارير وإحصاءات الأداء برسوم بيانية.
- **المساعد** — بحث في المصادر (مع ذكر المصدر) + حل مسائل الرياضيات (نص/صورة).
- **المحادثات** — تواصل مباشر بين المعلمين وطلبتهم المخصّصين والإدارة، بصلاحيات حسب الدور والمستوى.
- **الروضة** — ركن تعليمي غني ومستقل تمامًا: بطاقات ناطقة (اضغط لتسمع النطق بالعربية عبر
  محرّك النطق في المتصفح) للحروف الـ٢٨ والأرقام والألوان والأشكال والفواكه والحيوانات، وركن
  **«عُمان بَلَدي»** (العَلَم، الجَمَل، النخلة، الخنجر، القلعة…) وركن الآداب والكلمات، بالإضافة
  إلى ألعاب أسئلة ممتعة. (الرابط الأصلي للروضة كان معطوبًا، فبُني المحتوى تفاعليًا بالكامل.)
- **المستخدمون** — إدارة الأدوار والموافقات، **تخصيص المستويات/النطاق**، ربط أولياء الأمور بالطلبة،
  إعادة تعيين كلمة المرور، روابط الدعوة، و**مصفوفة صلاحيات الأدوار**.

**مزايا عامة:** بيانات المستخدم في الزاوية العلوية اليمنى، تبديل نظام الأرقام (عربية ١٢٣ / هندية 123)
لكامل التطبيق، تصميم متجاوب كامل للجوال، وحدّ رفع للملفات حتى ~50MB (استخدم روابط YouTube/Drive
للفيديوهات الأكبر).

## التشغيل محليًا
```bash
cd OLS
node server.js          # ثم افتح http://localhost:3000
```
أول حساب يُسجَّل يصبح **مدير النظام** تلقائيًا. الطالب يسجّل ويختار **صفّه الدراسي**، ويبقى
«بانتظار اعتماد المدير» — بعد الاعتماد يرى محتوى صفّه فقط (حصص، مناهج، اختبارات، تمارين).
المعلمون يُخصَّصون لصفوفهم من صفحة المستخدمين، أو ينضم الجميع عبر روابط دعوة من المدير.

## تفعيل المساعد الذكي (اختياري)
المساعد والبحث يعملان بدون مفتاح (بحث محلي). لتفعيل الإجابات الذكية وحل صور المسائل:
```bash
OLS_ANTHROPIC_KEY=sk-ant-...  node server.js
```
المفتاح يبقى على الخادم فقط ولا يصل إلى المتصفّح (المتصفّح ينادي `/api/assist`).

## البيانات والمزامنة
- الحسابات والمحتوى المشترك تُحفظ في مجلد خارج المستودع (`OLS_DATA_DIR` ← `~/ols-data` ← `./data`)
  حتى لا تُمحى عند إعادة النشر.
- المزامنة كل 12 ثانية عبر `/api/state`، والوسائط عبر `/api/blob` و`/api/file`.

## المصادر
كتب المنهج مصدرها الرسمي (وزارة التربية والتعليم / omanedubooks.com) وتُفتح من هناك حفاظًا على
حقوق النشر — التطبيق لا يستضيف ملفات الكتب المحمية. الروابط الأصلية في `OLS-Links.docx`.

## النشر
مطابق لـ RH ProjectHub: ارفع الملفات، اضبط `package.json`'s start = `node server.js`،
وشغّل الخادم. تحقّق من الإصدار الحيّ عبر `/api/version`.
